# Redbook Dashboard - Makeup

## 中文简介
本项目基于小红书美妆品牌相关数据，构建了一个轻量级前端可视化分析看板，用于展示数据处理、界面设计与交互式分析的综合能力。项目采用 HTML、CSS 和 JavaScript 实现，并通过 GitHub Pages 进行部署，适合作为数据可视化与前端展示类作品集项目。

## English Introduction
Based on Xiaohongshu beauty-brand-related data, this project develops a lightweight front-end analytics dashboard to demonstrate integrated skills in data handling, interface design, and interactive visualization. It is built with HTML, CSS, and JavaScript, and deployed via GitHub Pages as a portfolio-style data visualization project.
